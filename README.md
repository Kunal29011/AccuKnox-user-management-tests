# AccuKnox User Management Tests

## Project Overview

This project contains end-to-end automation test cases for OrangeHRM User Management module using Playwright and JavaScript.

## Test Scenarios Covered

* Login to OrangeHRM
* Navigate to Admin Module
* Add New User
* Search User
* Edit User Details
* Delete User

# Tech Stack

* Playwright
* JavaScript
* Node.js

## Setup Instructions

### Install Dependencies

npm install

### Install Browsers

npx playwright install

## Run All Tests

npx playwright test

## Run Individual Tests

npx playwright test tests/login.spec.js

## Playwright Version

1.60.0
